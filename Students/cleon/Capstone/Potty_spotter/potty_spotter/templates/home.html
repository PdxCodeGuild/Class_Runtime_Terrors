Kenapa Pilih potty_spotter?

{% spaceless %} Gratis Pendaftaran. Biaya les Termurah. Diskon paket untuk siswa baru plus voucher belajar 50 ribu. Transparansi keberlangsungan pertemuan les dan full Support lewat aplikasi potty_spotter Student. Tutor attendance code disetiap pertemuan les dan dapatkan tambahan point setiap kali kirim attendance code ke tutor. Semakin banyak point, semakin besar tambahan diskon untuk paket les berikutnya. Jadwal les fleksibel. Belajar apa saja dan kapan saja. Sesukamu! Tutor kompeten dan berpengalaman. Tutor yang mengajar dipastikan telah lulus tahapan seleksi administrasi, akademik tertulis dan demo mengajar dari potty_spotter. Mau tutor laki - laki atau perempuan? Bisa request. Jaminan ganti tutor jika tidak cocok, sampai siswa puas dan cocok. {% endspaceless %}

Kenapa memilih potty_spotter?


✔ Free registrasi
✔ Waktu belajar fleksibel. Bebas tentukan sendiri!
✔ Sistem belajar one-on-one. Siswa lebih fokus, konsentrasi meningkat, dan pembelajaran terarah/sesuai target.
✔ Pendalaman konsep materi. Diberikan juga latihan soal/pr/tugas dan persiapan sukses semua ujian sekolah.
✔ Belajar di rumah sendiri. Nyaman, kondusif dan terpantau.
✔ Tutor muda, berpengalaman dan kompeten.
✔ Boleh pilih tutor sendiri. Sesuai kriteria dan kebutuhan belajar siswa.
✔ Bebas konsultasi pelajaran di luar les privat.
✔ 1 sesi belajar maksimal 2 jam. Boleh tambah sesi melalui kesepakatan.
✔ Laporan evalusi siswa di akhir bulan.
✔ Pembayaran via transfer ke rekening potty_spotter.
